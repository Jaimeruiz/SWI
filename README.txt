-El proyecto es una plataforma para el alquiler de viviendas destinado principalmente a estudiantes de erasmus o de ciudades diferentes a las de su origen.

-Para desplegar el proyecto es necesario descargarlo en primer lugar, seguido abrir XAMPP CONTROLER PANEL para poder administrar la conexion con la base de datos.

-Para finalizar, abrir la consola de gitBash y acceder a la carpeta donde se ha alojado el proyecto dentro de nuestro equipo. Una vez estemos en la ruta ejecutar "npm start" y arrancara nuesrto proyecto.

-Por ultimmo abrir el navegador y escribir en la barra de busqueda "localhost:3000".
